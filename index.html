Kode Program Kivy

from kivy.app import App
from kivy.uix.button import Button
from kivy.uix.boxlayout import BoxLayout

class MyApp(App):
    def build(self):
        b_layout = BoxLayout(orientation='vertical')

        btn1 = Button(text='btn1',
                      size_hint=(0.5, 1),
                      pos_hint={'center_x': 0.5})
        btn2 = Button(text='btn2')
        btn3 = Button(text='btn3')
        btn4 = Button(text='btn4')

        b_layout.add_widget(btn1)
        b_layout.add_widget(btn2)
        b_layout.add_widget(btn3)
        b_layout.add_widget(btn4)

        return b_layout
MyApp().run()

Penjelasan Kode

Kode di atas adalah contoh program sederhana menggunakan framework Kivy di Python untuk membuat antarmuka pengguna grafis (GUI).

1. Impor Modul yang Dibutuhkan:

  • from kivy.app import App
    Mengimpor kelas App, yang merupakan fondasi untuk setiap aplikasi Kivy. Semua aplikasi Kivy harus mewarisi dari kelas ini.
  • from kivy.uix.button import Button
    Mengimpor kelas Button, yang digunakan untuk membuat widget tombol yang dapat diklik dalam antarmuka pengguna.
  • from kivy.uix.boxlayout import BoxLayout
    Mengimpor kelas BoxLayout, sebuah widget tata letak (layout) yang mengatur widget-widget anak-anaknya secara linear, baik secara horizontal maupun vertikal.

2. Definisi Kelas Aplikasi:

  • class MyApp(App):
    Mendefinisikan kelas aplikasi utama bernama MyApp yang mewarisi dari App. Ini adalah titik masuk untuk logika aplikasi Kivy Anda.

3. Metode build():

  • def build(self):
    Ini adalah metode inti dari kelas App yang harus ditimpa. Kivy memanggil metode ini saat aplikasi dimulai. Tugasnya adalah mengembalikan widget "akar" (root widget) dari aplikasi, yaitu widget teratas yang akan menampung semua elemen UI lainnya.
  • b_layout = BoxLayout(orientation='vertical')
    Membuat instance dari BoxLayout. Argumen orientation='vertical' menentukan bahwa widget-widget yang ditambahkan ke tata letak ini akan disusun dari atas ke bawah.

4. Pembuatan Tombol (Buttons):

  • btn1 = Button(text='btn1', size_hint=(0.5, 1), pos_hint={'center_x': 0.5})
    Membuat tombol pertama dengan teks "btn1".
    • size_hint=(0.5, 1): Ini adalah hint ukuran. Lebar tombol akan menjadi 50% (0.5) dari lebar ruang yang tersedia untuknya, dan tingginya akan menjadi 100% (1) dari tinggi ruang yang tersedia.
    • pos_hint={'center_x': 0.5}: Ini adalah hint posisi. Ini mencoba memposisikan pusat horizontal tombol pada 50% (tengah) dari lebar ruang yang tersedia. Dengan size_hint yang ditentukan, ini akan memusatkan tombol secara horizontal dalam slotnya di BoxLayout.
  • btn2 = Button(text='btn2')
    Membuat tombol kedua dengan teks "btn2". Karena tidak ada size_hint atau pos_hint yang ditentukan, tombol ini akan menggunakan nilai default size_hint=(1, 1), yang berarti akan mengisi seluruh ruang yang dialokasikan oleh BoxLayout untuknya secara proporsional.
  • btn3 = Button(text='btn3')
    Membuat tombol ketiga, serupa dengan btn2.
  • btn4 = Button(text='btn4')
    Membuat tombol keempat, serupa dengan btn2 dan btn3.

5. Menambahkan Widget ke Tata Letak:

  • b_layout.add_widget(btn1)
    Menambahkan btn1 ke dalam b_layout. Widget akan ditambahkan berurutan sesuai dengan urutan pemanggilan add_widget().
  • b_layout.add_widget(btn2)
    Menambahkan btn2.
  • b_layout.add_widget(btn3)
    Menambahkan btn3.
  • b_layout.add_widget(btn4)
    Menambahkan btn4.

6. Mengembalikan Widget Akar:

  • return b_layout
    Metode build() mengembalikan b_layout. Ini menjadikan b_layout sebagai kontainer utama (root widget) untuk seluruh aplikasi Kivy.

7. Menjalankan Aplikasi:

  • MyApp().run()
    Baris ini membuat instance dari kelas MyApp (aplikasi Anda) dan kemudian memanggil metode run() pada instance tersebut. Metode run() adalah yang memulai siklus hidup aplikasi Kivy, menampilkan jendela, dan menangani semua peristiwa (event) seperti klik mouse, sentuhan, dll. Tanpa baris ini, aplikasi tidak akan pernah terlihat atau berfungsi.

Tampilan yang Diharapkan:

Ketika kode ini dijalankan, Anda akan melihat sebuah jendela aplikasi dengan empat tombol yang tersusun secara vertikal. Tombol pertama ("btn1") akan dipusatkan secara horizontal dan hanya mengambil setengah lebar yang tersedia, sementara tiga tombol lainnya ("btn2", "btn3", "btn4") akan mengambil lebar penuh yang dialokasikan oleh BoxLayout dan masing-masing menempati bagian yang sama dari sisa tinggi vertikal.
